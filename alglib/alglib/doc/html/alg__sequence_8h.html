最长公共子序列 LCS(Longest Common Subsequence)，元素无需相邻

参数
[in]x第1个序列
[in]y第2个序列
[out]result输出最长公共子序列
返回
最长公共子序列长度
注解
动态规划，递推公式：
c[i][j] = 0 (i=0, or j=0)
c[i][j] = c[i-1][j-1] + 1 (i,j>0; x[i] = y[j])
c[i][j] = max{c[i][j-1], c[i-1][j]} (i,j>0; x[i] != y[j])

最大子数组 maximum subarray，要找出由数组成的一维数组中和最大的连续子数组，时间复杂度O(n)

参数
[in]n待分析数组长度
[in]a待分析数组
返回
最大子数组的和，全负数则返回0
注解
比如{5,-3,4,2}的最大子序列就是 {5,-3,4,2}，它的和是8,达到最大；而{5,-6,4,2}的最大子序列是{4,2}，它的和是6
