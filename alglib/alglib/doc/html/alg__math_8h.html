如果GCD(a,b) = d, 则存在x, y, 使d = ax + by; extended_euclid(a, b) = ax + by
二元一次不定方程ax+by=c，有解的重要条件是(a,b)|c；如果x0 y0 是一组解，则全部解：
x = x0 + b/(a,b) * t, y = y0 - a/(a,b) * t, (t属于Z)
